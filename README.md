# Microsoft AI Tour Tokyo 2026

[LTG189-JP: Microsoft 365 Agents SDK と Microsoft 365 Copilot API で実現するカスタムエージェント統合の実践](https://aitour.microsoft.com/flow/microsoft/tokyo26/sessioncatalog/page/sessioncatalog/session/1764020745222001E2cd) のサンプル デモです。

Microsoft 365 Agents SDK と Microsoft 365 Copilot API を使用して、カスタムの ToDo アプリにエージェント機能を提供します。ユーザーは Microsoft 365 で受信したメールから新しいタスクを追加できるようになります。

- [Microsoft 365 Agents SDK](https://learn.microsoft.com/ja-jp/microsoft-365/agents-sdk/agents-sdk-overview)
- [Microsoft 365 Copilot API](https://learn.microsoft.com/ja-jp/microsoft-365-copilot/extensibility/copilot-apis-overview)
